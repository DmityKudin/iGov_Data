Дані по зверненню на видачу довідки про величини фонових концентрацій забруднювальних речовин в атмосферному повітрі №[sID_Order]
Назва організації, яка запитує довідку - [sOrgName].
Адреса організації, яка запитує довідку - [sOrgAdress].
Назва населеного пункта, для якого відбувається запит - [sCity].
Назва підпрємства, для якого запитуються величини фонових концентрацій - [sCompName].
Адреса підприємства - [sCompAdress].
Тип підприємства - [asCompType].
Перелік речовин, які викидаються підприємством - [sCompElem].
Бажаний спосіб отримання довідки - [asGetType].
Дата подання заявки - [sDateCreateProcess].
